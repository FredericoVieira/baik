# Baik

Baik is a webplatform to help users to find the nearest bikes to use!

### Prerequisites:

yarn

### Testing

To run tests:

```
yarn test
```

### Running:

To set this projetct locally, first install dependencies:

```
yarn install
```

To run:

```
yarn start
```

or build to production:

```
yarn build
```

Access localhost:3000
